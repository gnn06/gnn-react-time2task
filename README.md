# Time2Task
*Organiser votre temps pour réaliser vos tâches.*

Time2Task n'est ni un agenda (Outlook fait ça très bien), ni un gestionnaire de tâche (Jira fait ça très bien).
Avec Time2Task vous affectez vos tâches à des créneaux de temps. Ces créneaux de temps vous permettez d'être concentré 
pour traiter des tâches similaires (time blocking). Vous planifiez votre travail dans une approche "par entonnoir" ; 
d'abord au niveau des créneaux les plus gros (le mois, la semaine) puis des créneaux plus petits (la journée, l'heure).
